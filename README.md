# MERN To-Do App

A full-stack To-Do List application built using the MERN stack.

## Tech Stack
- MongoDB
- Express.js
- React (Vite)
- Node.js

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- MongoDB database storage

## Installation

1. Clone the repository
git clone https://github.com/swadhasinha2004/mern-todo-app.git

2. Install dependencies

Backend:
cd Backend
npm install

Frontend:
cd Frontend
npm install

3. Run the application

Backend:
node index.js

Frontend:
npm run dev
